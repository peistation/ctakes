[[cd_padtermspotter]]
=== PAD Term Spotter (optional) ===
The PAD term spotter pipeline processes radiology note textual extractions specifically pertaining to the diagnosis, treatment, etc. of lower
limb Peripheral Artery Disease (PAD) (e.g. stenosis/occlusion paired with popliteal/femoral).  
It classifies of each radiology-related document with respect to the presence of PAD. 

Further documentation is in +{inst-root-dir}/PAD term spotter/README+.
